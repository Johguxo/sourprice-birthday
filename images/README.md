# 🌸 Carpetas de imágenes

Coloca aquí las fotos para tu web de cumpleaños. Cada carpeta corresponde a una sección distinta.

## 📁 `hero/`
Foto **principal** que aparece junto al título "Feliz Cumpleaños".
- Archivo esperado: `hero.jpg`
- Recomendado: foto vertical (3:4), por ejemplo `1200x1600 px`.

## 📁 `carousel/`
Fotos del **carousel** "Nuestros momentos".
- Archivos esperados: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg` (puedes agregar más editando `index.html`).
- Recomendado: horizontales (16:10), por ejemplo `1600x1000 px`.

## 📁 `gallery/`
Fotos para la galería tipo **polaroid** "Pequeños recuerdos".
- Archivos esperados: `1.jpg` a `6.jpg`.
- Recomendado: cuadradas (1:1), por ejemplo `800x800 px`.

---

### 💡 Notas
- Si una imagen no existe, la sección muestra un placeholder rosado bonito.
- Puedes usar `.jpg`, `.png` o `.webp` (cambia la extensión también en `index.html`).
- Para cambiar los textos / mensajes, edita directamente `index.html`.
